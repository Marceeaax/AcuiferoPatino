        Subir puntos de muestreo

        Elegí cómo querés cargar tus puntos.

        La opción manual mantiene el flujo actual de elegir una ubicación en el mapa.
        Carga masiva de puntos por CSV
        Formato estricto requerido
        El archivo debe ser un CSV o TSV con encabezados en la primera fila. Para que la importación funcione correctamente, respetá la estructura de columnas y el sistema de coordenadas indicado.

        Columnas obligatorias
        El archivo debe incluir como mínimo:
        x
        y
        Además, se recomienda incluir al menos una columna descriptiva:
        codigo_pozo o nombre_lugar

        Columnas aceptadas
        El formato recomendado es usar nombres como:
        codigo_pozo, nombre_lugar, x, y, Fecha_muestreo
        N-Amoniacal (mg/L), N-Nitritos (mg/L), N-Nitratos (mg/L)
        Alcalinidad Total (mg/L), Materia Organica (mg/L), Conductividad (uS/cm), pH
        Bicarbonato (mg/L), Carbonato (mg/L), Sulfato (mg/L), Magnesio (mg/L)
        Calcio (mg/L), Sodio (mg/L), Potasio (mg/L), Cloruro (mg/L)
        Arsenico (mg/L), Mercurio (mg/L), Manganeso (mg/L), Cobre (mg/L)
        Cromo Total (mg/L), Coliformes Fecales (UFC/100 mL)

        Reglas de formato
        Las coordenadas deben ser numéricas.
        La primera fila debe contener solo encabezados.
        Cada fila siguiente debe representar un único punto.
        Las columnas no incluidas o las celdas vacías se guardarán como NULL.
        No uses celdas combinadas, títulos extra, notas al pie ni filas vacías antes del encabezado.

        Sistema de coordenadas
        Si el archivo usa coordenadas UTM como tus planillas históricas, elegí EPSG:32721.
        Si el archivo ya está en longitud/latitud GPS, elegí EPSG:4326.

        Recomendación práctica
        La forma más segura es reutilizar uno de los archivos _para_subir.csv ya generados, manteniendo exactamente los mismos encabezados.

        Vista previa técnica
        🧭 Cargar capa GeoJSON

        ¿Qué es una capa GeoJSON?
        Es un archivo que representa información geoespacial, como polígonos, puntos o líneas. Es utilizado ampliamente en sistemas de información geográfica (GIS).

        🗺️ Sistema de Coordenadas Esperado
        Este visor trabaja con el sistema de coordenadas EPSG:4326, también conocido como WGS 84, el estándar GPS global. Asegurate de que tus coordenadas estén en latitud y longitud.

        🔁 ¿Cómo se guarda la capa?
        Cada geometría del archivo se transforma en un objeto MultiPolygon y se guarda en la base de datos en la tabla patino, usando PostGIS. Este modelo requiere geometría válida (cerrada, sin errores topológicos).

        📂 Seleccioná tu archivo

        Shapefile (.zip)
        Si tenés la capa en formato Shapefile, podés subir un .zip que incluya al menos los archivos .shp, .shx, .dbf y .prj.

        Conversión automática
        El sistema reproyecta la capa a EPSG:4326 antes de guardarla en la base.
